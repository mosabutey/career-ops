# Career-Ops LifeSci for Codex

Read `CLAUDE.md` for the main project instructions. It is the source of truth for routing, workflow, and behavior.

For Codex-specific setup and mode routing, see `docs/CODEX.md`.

Key rules:
- Reuse the existing modes, scripts, templates, and tracker flow instead of creating parallel automation.
- Keep user-specific customization in `config/profile.yml`, `modes/_profile.md`, `article-digest.md`, or `portals.yml`.
- Never put user-specific content in `modes/_shared.md`.
- Never submit an application on the user's behalf.
