# Example: Physician-Scientist to Medical Affairs

This fictional example shows how to configure Career-Ops for a physician-scientist targeting:
- MSL roles
- medical affairs manager roles
- scientific affairs roles
- translational or clinical scientist roles

Recommended stage path:
- `experienced_professional` or `advanced_training`, depending on the person

Primary role pack:
- `biopharma_medical`

Secondary role packs:
- `healthtech_scientific`
- `life_sciences_consulting`
