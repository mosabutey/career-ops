# Example: Graduate Student to Life Sciences Consulting

This fictional example shows how to configure Career-Ops for a graduate student targeting:
- life sciences consulting internships
- healthcare strategy internships
- full-time associate roles after graduation

Recommended stage path:
- `student_early`

Primary role pack:
- `life_sciences_consulting`

Secondary role packs:
- `adjacent_generalist`
- `healthtech_scientific`
